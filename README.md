# LadderTop

**バージョン: v1.3.0**

s2e-ladder（梯子登りゲーム）の**頂上（ゴール上空）へテレポート**する補助プラグインです。`/ladder tp` の初期位置の真上、**ゴール中央**へ飛ばします。動作確認やゴール演出の用途で作成しています。

> ⚠️ s2e-ladder 専用です。

---

## 概要・仕組み

`/ladder tp` の初期位置は s2e-ladder の `plugins/s2e-ladder/config.yml` の `ladder.start` に保存されています。本プラグインはこのファイルから初期位置とゴール高さを**自動取得**し、その真上（ゴール中央）へテレポートします。

```yaml
# s2e-ladder/config.yml（参照元）
ladder:
  height: 1000                 # ゴールまでの高さ
  start: { world: world, x: ..., y: 20.0, z: ..., yaw: 0.0, pitch: 0.0 }
```

**頂上の Y（デフォルト）= `ladder.height` ＋ `extra-height`（絶対 Y）**
- 例: `height=1000`, `extra-height=40` → 頂上 **Y = 1040**
- `start.y` は**加算しません**（絶対 Y）。高さ引数を渡すとその値を**絶対 Y**として使います。
- 余白（+20）はゴールのわずか上空に出して着地させるためのマージンです。

**着地 X/Z = ゴール中央（7×7）**
- `ladder.start` の真上で X/Z を**ブロック中央(floor+0.5)へ自動スナップ**します。
- ゴールは **7×7** でスタートが端にあるため、config で **`center-x-offset: -3.0`** を入れて X を中央へ寄せています（X = 627.5 - 3 = **624.5**、Z = **640.5**）。
- ゴール構造を動かした場合は `center-x-offset` / `center-z-offset` で微調整します。

向き（yaw/pitch）は初期位置の値を引き継ぎます。

**タイトル表示（v1.2.0〜）**
- テレポートした対象プレイヤーへ画面中央に**タイトル**を表示します（既定: `頂上テレポート`、サブタイトルなし）。
- 文言・表示秒数は config の `title:` で変更できます（`%y%` は頂上 Y に置換）。

---

## コマンド

```
/laddertop [高さ] [対象]      （エイリアス: /ltp）
```

| コマンド | 説明 |
|---|---|
| `/laddertop` | 実行者を頂上へTP（デフォルト Y = height + extra-height = 1040） |
| `/laddertop 1500` | 頂上の Y（絶対値）を 1500 に指定してTP |
| `/laddertop 1500 <名前>` | 指定プレイヤーをTP |
| `/laddertop 1500 all` | オンライン全員をTP |
| `/laddertop 1500 random` | ランダムな1人をTP |
| `/laddertop <名前\|all\|random>` | デフォルト高さで対象をTP（高さ省略） |
| `/laddertop reload` | `config.yml` を再読み込み |

- 第1引数が**数値なら Y（絶対値）**、**非数値なら対象**として解釈します。
- 着地 X/Z は常に**ゴール中央（ブロック中央）へ自動スナップ**します。
- 権限: `laddertop.use`（デフォルト op）
- タブ補完対応（高さ候補 / 対象）

---

## config.yml

```yaml
# ゴール高さ(s2e の ladder.height)に加える余白ブロック数（着地用の上空マージン）。
# 頂上 Y(デフォルト) = ladder.height + extra-height（例: 1000 + 40 = 1040）。
extra-height: 40

# 着地座標の微調整（ブロック中央スナップ後に加算するオフセット）。
# ゴールは 7×7 でスタートが端にあるため、X を -3 ブロックして中央へ寄せる。
center-x-offset: -3.0
center-z-offset: 0.0

# テレポート時に表示するタイトル（サブタイトルなし。%y%=頂上Y、§x=色/装飾、20ticks=1秒）。
title:
  main: '§6§l頂上テレポート'
  fade-in-ticks: 10
  stay-ticks: 50
  fade-out-ticks: 20
```

## インストール / 反映

1. ビルドした `LadderTop.jar` を `plugins/` に配置。
2. サーバー再起動（または `/reload confirm`）で有効化。`config.yml` は初回起動時に自動生成。

ビルドは [minecraft-build-env] のオフライン手順。

---

## 注意点

- 初期位置・ゴール高さは s2e-ladder の **config.yml ファイルから読み取り**ます（リフレクション不使用）。`ladder.start` が未設定だとエラーになります。
- 頂上 Y はワールドの建築上限（例: 320）を超えますが、プレイヤーは上限より上にも存在できるため問題ありません。
